## Problem

Create a program with a Student class that stores a student’s name, ID, grades (percentage), and credits. Include appropriate constructors, accessors, mutators, encapsulation, a static variable to count students, an honor roll check (grades ≥ 95), and a recursive method to calculate the factorial of credits. Then, create a subclass HighSchoolStudent that adds a gradeLevel property and overrides the honor roll check (grades ≥ 90). In your Main class, create and test both types of students, and display their information and the total number of students created.

## Objective
Covering void, return, public, private, static, accessor and mutator methods, constructors, static variables, “this” keyword, encapsulation, inheritance and recursion.
