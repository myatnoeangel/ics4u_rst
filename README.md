# Java Programming Concepts - ICS4U RST

This repository contains a collection of Java programming exercises and concept summaries developed as part of the ICS4U Computer Science course (Ontario Secondary School Diploma).

The goal of this assignment was to create concise summaries and programming problems demonstrating major Java concepts. Each section includes a problem description and a working solution implemented in Java.

These exercises reflect my early learning in object-oriented programming, logical reasoning, iteration, and data structures.

---

## Topics Covered
This project explores the following core Java topics:
- Classes and Objects
- Constructors and Methods
- Accessor and Mutator Methods
- Encapsulation
- Static Variables
- Inheritance and Method Overriding
- Recursion
- Boolean Expressions and De Morgan’s Law
- Conditional Statements
- Iteration (for loops, while loops, nested loops)
- Arrays and ArrayLists

---

## Learning Outcomes
Through this project I practiced:
- designing object-oriented programs in Java
- applying boolean logic and conditional statements
- implementing iteration using multiple loop structures
- working with arrays and dynamic collections (ArrayLists)
- writing structured and reusable code
